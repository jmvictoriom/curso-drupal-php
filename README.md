# Curso Drupal + PHP: De Cero a Experto

## Como usar este curso

Cada leccion tiene su propia carpeta numerada con:

- `teoria.php` — Explicaciones + ejemplos que puedes ejecutar
- `ejercicio.php` — Retos para que practiques
- `solucion.php` — Solucion (no espiar antes de intentarlo, no todas las lecciones la tienen)

### Para ejecutar cualquier archivo:

```bash
php nombre-del-archivo.php
```

### Para pedir que revise tu ejercicio:

Simplemente dime "revisa mi ejercicio de la leccion X" y te doy feedback.
